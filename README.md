### 🎁 **Fiture**

- App Name
- Todo Function It Self

### 📄 **Page Include**

### 🎖  **Installing**
```
dependencies:
  flutter:
    sdk: flutter
  cupertino_icons: ^1.0.2
  flutter_slidable: ^1.2.0
  google_fonts: ^3.0.1
```

### 🚀 **Getting Started**
1. Clone this project || Download Zip from GitHub
2. Run `flutter pub get` In Terminal
3. Run the project In Web or Emulator
